# /싱크업요약

## 목적
싱크업 시트에서 교육 과정 개요를 추출하여 한 눈에 볼 수 있는 요약을 생성한다.
빈 필드나 내용이 부족한 항목을 경고로 표시한다.

## 입력
- `syncup-sample.csv` (로컬 CSV) 또는
- [학습용] 싱크업 Sheets (MCP, M5 이후 사용 가능)

## 실행 절차

1. 데이터 소스 탐색
   - 먼저 `samples/syncup-sample.csv` 또는 `my-data/` 에서 싱크업 CSV 검색
   - CSV가 없고 MCP가 연결되어 있으면 학습용 싱크업 Sheets에서 읽기
2. 핵심 정보 추출
   - 과정명, 교육형태, 교육일정, 교육장소, 교육대상
   - 담당자 정보: 기업 담당자(이름/이메일/전화), FC 담당(LDM/LXM/LM)
   - 강사명, 강사이메일, 조교명
3. 빈 필드 감지
   - 비어있거나 내용이 부족한 필드를 ⚠️ 경고로 표시
   - 필수 필드 목록: 과정명, 교육일정, 강사명, 기업담당자_이름
4. 요약 카드 출력

## 출력 형식

```
📋 교육 개요 요약
━━━━━━━━━━━━━━━━
과정명: {과정명}
교육형태: {교육형태}
일정: {교육일정}
장소: {교육장소}
대상: {교육대상}

[담당자]
기업: {기업담당자_이름} ({기업담당자_이메일})
FC: LDM {FC_LDM} / LXM {FC_LXM} / LM {FC_LM}
강사: {강사명} ({강사이메일})
조교: {조교명}

⚠️ 빈 필드 경고:
- {빈 필드 1}
- {빈 필드 2}
```

## 여러 과정이 있을 때
CSV에 여러 행(과정)이 있으면, 각 과정별로 요약 카드를 생성한다.
빈 필드가 많은 과정은 "⚠️ 싱크업 보완 필요" 표시.
