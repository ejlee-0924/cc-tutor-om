# OM 용어 사전 (비유 포함)

> 기술 용어를 OM 업무 맥락으로 풀어서 설명합니다.

## Claude Code 기본

| 용어 | OM 비유 | 설명 |
|------|---------|------|
| **Claude Code** | 현장 출장 전문가 | 내 컴퓨터에서 직접 파일을 읽고 수정하는 AI |
| **Gems/GPTs** | 전화 상담 | 브라우저 안에서만 동작하는 AI |
| **터미널** | 명령어 입력 창 | 텍스트로 컴퓨터에 지시하는 창 |
| **CLI** | 텍스트 기반 인터페이스 | GUI(마우스 클릭)의 반대 개념 |
| **Plan 모드** | 먼저 계획서 확인 후 실행 | 복잡한 작업을 계획→실행 분리하는 모드 |

| 용어 | OM 비유 | 설명 |

| **CSV** | 엑셀 호환 텍스트 파일 | 쉼표로 구분된 데이터 파일 |

## 검수 프로토콜 (박규은 모델)

| 용어 | 의미 |
|------|------|
| **PASS** | 수정 불필요 (Fatal 0건) |
| **CONDITIONAL** | 경미한 수정 (Fatal 0건, Quality 1~3건) |
| **REJECT** | 즉시 수정 필수 (Fatal 1건 이상) |
| **Fatal Error** | 치명적 오류 (내용 오류, 기업명 오타, 누락 섹션) |
| **Quality Issue** | 품질 개선점 (톤앤매너, 여백, 줄간격) |
