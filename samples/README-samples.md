# samples/ 폴더 안내

이 폴더에는 실습용 샘플 데이터가 들어 있습니다.

## 파일 목록

| 파일 | 활용 모듈 | 설명 |
|------|----------|------|
| survey-sample.csv | 2, 7 | 가상 만족도 설문 30건 (생성형 AI 로우코딩 과정) |
| apps-script-broken.js | 2 | 의도적 오류 5가지가 있는 앱스크립트 (강사 데이터 동기화) |
| slide-sample.pptx | 2 | 의도적 오탈자 5개 포함 교안 (10슬라이드) |
| curriculum-sample.pdf | 2, 4 | 가상 AI 교육 커리큘럼 (5모듈, 6페이지) |
| resume-sample.pdf | 4 | 가상 강사 이력서 3건 (매칭 점수 차이 설계) |

## 사용법

1. 각 모듈 실습 시 Claude Code가 이 폴더의 파일을 자동으로 안내합니다
2. 실제 업무 데이터가 있으면 `my-data/` 폴더에 넣어주세요 (더 효과적)
3. `my-data/`에 데이터가 없으면 이 폴더의 샘플로 실습합니다

## ⚠️ 주의

- 이 데이터는 **완전히 가상**입니다 (실제 인물/과정 아님)
- apps-script-broken.js에는 **의도적으로 5가지 버그**가 포함되어 있습니다
- slide-sample.pptx에는 **의도적으로 5개 오탈자**가 포함되어 있습니다
